## 📌 Overview

A deep-learning-based resume optimization and ATS matching system that analyzes resumes, evaluates their compatibility with job descriptions, and generates improved versions using AI.

---

## 🔥 Key Highlights

- Embedding-based semantic matching  
- Cosine similarity scoring  
- GenAI resume optimization  
- Kaggle Resume Dataset (~200 resumes)  
- +18% improvement  

---

## 🧠 Architecture

Resume + Job Description → Parsing → Preprocessing → Embeddings → Similarity → Scoring → GenAI → Output

---

## ⚙️ Model

v = f(text), v ∈ ℝⁿ  

Similarity:  
sim(A,B) = (A·B)/(||A||·||B||)

---

## 📉 Loss

L = 1 - cos_sim(A,B)

---

## 📊 Metrics

Accuracy, Precision, Recall, F1-score
